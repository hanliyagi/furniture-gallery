# Contributing

Thank you for contributing to Furniture Gallery.

## Before opening a pull request

1. Keep each change focused and include tests when behavior changes.
2. Run `npm run verify` and `npm run catalog:validate`.
3. Do not commit `.env.local`, generated `dist/` files, or `catalog-output/`.
4. Preserve the catalog schema and use meters for model dimensions.

## Reporting issues

Include the browser, operating system, reproduction steps, expected behavior, and screenshots when a visual issue is involved.
